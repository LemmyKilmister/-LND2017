Ejercicios sobre reset/Autoprefixer/font-awsome/fuentes

Que es un reset en CSS

1º) Son una serie de instrucciones que damos ya sea manualmente o con una aplicación web para anular las instrucciones css que de por si los navegadores traen por defecto. 2-)

Explicando un reset

2º) margin: 0; padding: 0; border: 0; font-size: 100%; font: inherit; // Poniendo estas instrucciones ponemos al 0 propiedades como el "margin" el "padding" y el "border" El"Font-size" especifica poner las dimensiones de la letra al 100%, el "font" en inherit para obligarlo a heredar propiedades.

Que es un autoprefixer

Un Autoprefixer es una aplicacion web o Plugin el cual facilita de forma facil utilizar prefijos de los motores de diseño en cualquier navegador.
